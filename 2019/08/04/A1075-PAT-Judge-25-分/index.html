1075 PAT Judge (25 分)

The ranklist of PAT is generated from the status list, which shows the scores of the submissions. This time you are supposed to generate the ranklist for PAT.

Input Specification:

Each input file contains one test case. For each case, the first line contains 3 positive integers, N (≤104 ), the total number of users, K (≤5), the total number of problems, and M(≤105), the total number of submissions. It is then assumed that the user id’s are 5-digit numbers from 00001 to N, and the problem id’s are from 1 to K. The next line contains K positive integers p[i](i=1, …, K), where p[i] corresponds to the full mark of the i-th problem. Then M lines follow, each gives the information of a submission in the following format:

user_id problem_id partial_score_obtained
where partial_score_obtained is either −1 if the submission cannot even pass the compiler, or is an integer in the range [0, p[problem_id]]. All the numbers in a line are separated by a space.

Output Specification:

For each test case, you are supposed to output the ranklist in the following format:

rank user_id total_score s[1] ... s[K]
where rank is calculated according to the total_score, and all the users with the same total_score obtain the same rank; and s[i] is the partial score obtained for the i-thproblem. If a user has never submitted a solution for a problem, then "-"must be printed at the corresponding position. If a user has submitted several solutions to solve one problem, then the highest score will be counted.

The ranklist must be printed in non-decreasing order of the ranks. For those who have the same rank, users must be sorted in nonincreasing order according to the number of perfectly solved problems. And if there is still a tie, then they must be printed in increasing order of their id’s. For those who has never submitted any solution that can pass the compiler, or has never submitted any solution, they must NOT be shown on the ranklist. It is guaranteed that at least one user can be shown on the ranklist.

题意

唉，这道题不想让人吐槽了，本来好好写了代码，一直错，快要放弃的时候，找到错误了，结果无论怎么改，都是两个测试点不过，只有21分，气的想砸电脑。后面参考了柳神的博客，才发现她写的真的简单，膜柳神 QAQ。不说废话了

  • 题意大概是，给你一个pat的数据，让你排名
  • 输入要求就不说了，输出有几个注意的点。
  • 1.如果编译不通过，输入会是-1。
  • 2.编译通过但是是0分也要输出
  • 3.按照总分递减顺序排名，如果总分相同，按照通过的题目数量递减排名，如果还是相同，按照id递增排序。
  • 4.如果没有回答任何题目或者回答了但是都是编译不通过的，也不会输出（非常重要）

思路

用一个结构体存储用户的排名，总分，id，是否要输出等信息，这里结构体内用了一个向量来存储每一题的分数，并且在输入分数的时候，实时更新这个分数。
对于输入的分数，只要不是-1，我们就将它和已有分数比较。而且记录当前结构体的show为true,因为如果一旦有-1的情况，也就是编译错误或者没有做过这个题目，那么除非全为这类，不然一定会覆盖-1。
如果输入的分数为-1，我们将其记为-2，也就是编译出错的，后来在输出的时候好判断输出符号。
然后就是基本的遍历操作了。

代码

#include<cstdio>
#include<algorithm>
#include<vector>
using namespace std;
struct user{
    int id,total=0,rank,passcount=0;//id 总分 排名 满分个数
    vector<int>score;
    bool show=false;//是否显示
};
bool cmp(user a,user b){//比较判断
    if(a.total!=b.total)return a.total>b.total;
    else if(a.passcount!=b.passcount)return a.passcount>b.passcount;
    else return a.id<b.id;
}
int main(){
    int uNum,pNum,n;//参加人数 题目个数 记录个数
    scanf("%d%d%d",&uNum,&pNum,&n);
    vector<int>full(pNum+1);//满分数组
    vector<user>Users(uNum+1);
    for(int i=1;i<=pNum;i++){
        scanf("%d",&full[i]);
    }
    for(int i=1;i<=uNum;i++){
        Users[i].score.resize(pNum+1,-1);//初始化
    }
    int id,pth,score;
    for(int i=1;i<=n;i++){
        scanf("%d%d%d",&id,&pth,&score);
        Users[id].id=id;
        Users[id].score[pth]=max(Users[id].score[pth],score);//实时更新分数
        if(score!=-1){
            Users[id].show=true;
        }else if(Users[id].score[pth]==-1){
            Users[id].score[pth]=-2;
        }
    }
    for(int i=1;i<=uNum;i++){
            for(int j=1;j<=pNum;j++){
                if(Users[i].score[j]!=-1&&Users[i].score[j]!=-2){
                    Users[i].total+=Users[i].score[j];//计算总分
                }
                if(Users[i].score[j]==full[j]){
                    Users[i].passcount++;//满分人数
                }
            }
        }
    sort(Users.begin()+1,Users.end(),cmp);
    for(int i=1;i<=uNum;i++){
        Users[i].rank=i;
        if(i!=1&&Users[i].total==Users[i-1].total){
            Users[i].rank=Users[i-1].rank;//计算排名
        }
    }
    for(int i=0;i<=uNum;i++){
        if(Users[i].show==true){
            printf("%d %05d %d",Users[i].rank,Users[i].id,Users[i].total);
            for(int j=1;j<=pNum;j++){
                if(Users[i].score[j]!=-1&Users[i].score[j]!=-2)
                printf(" %d",Users[i].score[j]);
                else if(Users[i].score[j]==-1)
                printf(" -");
                else printf(" 0");
            }
            printf("\n");
        }
    }
    return 0;



}
