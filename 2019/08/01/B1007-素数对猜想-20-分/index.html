1007 素数对猜想 (20 分)

让我们定义dn为:dn=pn+1−pn ，其中pi 是第i个素数。显然有d1 =1，且对于n>1有dn是偶数。“素数对猜想”认为“存在无穷多对相邻且差为2的素数”。

现给定任意正整数N(<105)，请计算不超过N的满足猜想的素数对的个数。

输入格式:

输入在一行给出正整数N。

输出格式:

在一行中输出不超过N的满足猜想的素数对的个数。

输入样例:

20

输出样例:

4

思路

第一次直接将题目范围内的所有素数全部存在数组prime中
第二次找到最后小于N的素数在prime中的下标k。例如19是小于20的最后一个素数，他在数组prime中的下标为7(此时prime为：2,3,5,7,11,13,17,19,...)
遍历前k个素数，计算素数对的个数count.
也可以在寻找n的素数的同时，计算两个相邻的素数相差是否为2，然后计数。

代码1：

#include<cstdio>
#include<cmath>
const int maxn=100000;
int prime[maxn],pNum=0;
bool p[maxn]={0};
void Find_Prime(){
    for(int i=2;i<maxn;i++){
        if(p[i]==false){//计算素数，通过哈希表法，为素数的哈希值为false
            prime[pNum++]=i;
            for(int j=i+i;j<maxn;j+=i){//因为在计算过程中，只要是素数的倍数，一定不会再是素数，从而减少计算量
                p[j]=true;
            }
        }
    }
}
int main(){
    Find_Prime();
    int n,k,count=0;
    scanf("%d",&n);
    for(int i=0;i<n;i++){
        if(prime[i]>n)//找到最后一个小于n的素数在prime中的下标
        {
            k=i-1;
            break;
        }
    }
    for(int i=0;i<k;i++){
        if(prime[i+1]-prime[i]==2)
        count++;
    }
    printf("%d",count);
    return 0;
}

代码2：

#include<iostream>
#include<cmath>
using namespace std;
int main()
{
    int n;
    cin>>n;
    int i,j,cnt=0;
    int pre=2;//记录前一个素数
    for(i=3;i<=n;i++)
    {
        for(j=2;j<=sqrt(i);j++)
        if(i%j==0)
        break;
        if(j>sqrt(i))
        {
            if(i-pre==2)
            cnt++;
            pre=i;
        }
    }
    cout<<cnt;    
    return 0;
}
